**SwasthyaAI**
AI Decision Intelligence Platform for Public Healthcare

SwasthyaAI is an AI-powered decision intelligence platform designed to improve public healthcare operations while helping citizens access the healthcare schemes they are eligible for.

**🗺️ User Flow
👥 Citizen Journey**
Open the SwasthyaAI portal.
Describe your profile in natural language.
JanMitra extracts relevant information.
The eligibility engine identifies matching schemes.
**View:**
Eligible schemes
Benefits
Required documents

**Application steps**
👨‍⚕️ Hospital Staff Journey->
Login using Staff Credentials->
Open JanMitra Staff Mode->
Enter patient information or use existing patient records.

**View:**
Eligible schemes,
Required documents,
Missing information,
Recommended next steps

After login:


**Dashboard**

Medicine Inventory,

Inventory Intelligence

View:

JanMitra Staff Portal

View:

Scheme eligibility results,

District AI Copilot

Ask questions such as:

Which PHCs need attention today?,
Which medicines may run out this week?,
Which hospitals require redistribution support?


The AI summarizes district-wide operational insights and recommendations.

## 🛠 Development Setup

Detailed setup instructions for running the project locally are available in the respective module documentation:

- `frontend/README.md` → Frontend setup, environment variables, and deployment instructions.
- `backend/README.md` → Backend setup, database configuration, API configuration, and deployment instructions.

This repository follows a modular structure so that frontend and backend services can be developed, deployed, and maintained independently.
